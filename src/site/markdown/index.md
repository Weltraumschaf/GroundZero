## Build and Install

A current [Maven][1] installation is required.

Clone this repository and change into in:

    $ git clone https://github.com/Weltraumschaf/GroundZero.git
    $ cd GroundZero

Build with maven:

    $ mvn clean install

Have fun:

    $ ./bin/groundzero -h

[1]: http://maven.apache.org/
